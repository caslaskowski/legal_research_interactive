Mixed Review — The Diagnostic Moves

Chapter drills tell you which skill to use before you start; practice never does. This deck interleaves the Part I diagnostic moves — classify the question, pick the source, rank the authority, spot the compound answer, fix the search — shuffled, so the first thing each item asks of you is recognizing which move it needs.

How to use this: the skill tag on each card stays hidden until you answer. The deck reshuffles on every visit; come back for spaced passes. Miss one? Its feedback names the chapter to revisit.
